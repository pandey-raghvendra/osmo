# Code of Conduct

osmo follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1.

## Summary

Be respectful. No harassment, discrimination, or personal attacks. Criticism of ideas is welcome; criticism of people is not.

## Scope

Applies in all project spaces: GitHub issues, PRs, discussions, and any venue where you represent osmo.

## Enforcement

Report violations privately via the contact listed in [SECURITY.md](SECURITY.md). All reports are confidential. Violations may result in a temporary or permanent ban.
